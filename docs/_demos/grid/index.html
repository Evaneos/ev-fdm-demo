Grid system for responsive panels

Contrary to the vast majority of responsive grid system developped, ev-grid is not based on the media screen query. Instead, the column collapse based its container size. This allow for much more dynamic comportement (for instance, some part of the page being collapsed while other are not).

The possibilities are the same as the bootstrap grid.

Example: Stacked to horizontal

Using a single set of .col-md-* grid classes, you can create a basic grid system that starts out stacked in extra small to small container before becoming horizontal on medium+ container. Place grid columns in any .row.
