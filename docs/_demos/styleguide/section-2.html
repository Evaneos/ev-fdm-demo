2.1 - Unstyled list

Remove the default user agent style on list with .list-unstyled

    • Eggs
    • Milk
    • Beers 
<ul class="list-unstyled">
	<li> Eggs </li>
	<li> Milk </li>
	<li> Beers </li>
</ul>

2.2 - Standard list

A standard list in ev-fdm. You can add the base class .ev-list to any div. All of its direct children will be styled like list element.

    • Eggs
    • Milk
    • Beers 
<ul class="ev-list">
	<li> Eggs </li>
	<li> Milk </li>
	<li> Beers </li>
</ul>

2.2.1 - Activable lists

For activable list, you can add the .ev-list-activable. This useful with the activable-set directive.

    • Eggs
    • Milk
    • Beers 
<ul class="ev-list ev-list-activable">
	<li activable> Eggs </li>
	<li activable class="active"> Milk </li>
	<li activable> Beers </li>
</ul>

2.3 - List separator

You can separate two part of a list by adding a .ev-separator-top or a .ev-separator-bottom class on a list element.

    • Eggs
    • Milk
    • Beers
    • Screwdriver
    • Hammer
    • Cat 
<ul class="ev-list">
	<li> Eggs </li>
	<li> Milk </li>
	<li class="ev-separator-bottom"> Beers </li>
	<li> Screwdriver </li>
	<li> Hammer </li>
	<li class="ev-separator-top"> Cat </li>
</ul>
