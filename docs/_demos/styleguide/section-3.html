3.0 Helper classes

3.1 Gray colors classes

Gray colors are neutral color used when the support should not convey any specific informations. They can be used ad background color or text colors

3.1.1 Gray background colors

A scale of gray background colors

  • .bg-gray

    Add a gray background
  • .bg-light

    Add a light background
  • .bg-lighter

    Add a lighter background
  • .bg-dark

    Add a dark background
  • .bg-darker
