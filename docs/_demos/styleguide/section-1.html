1.0 Buttons

A basic UI component triggering an action. Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

1.1 Base style & state

  • default

    A button in normal state
  • :hover

    Hover state
  • :active

    Active state
  • .disabled

    Disabled button. If you use <button> element, that is acceptable to add disabled attribute instead.

     
<button class="btn btn-primary {$modifiers}">Button</button>
<button class="btn {$modifiers}">Button</button>

1.2 Skins

You can change the color of the button by adding any of these classes

  • .btn-default

    A neutral button
  • .btn-danger

    A red button, generally used for emphasing dangerous action (with no come back possible)
  • .btn-primary

    A button from the primary color, generally used as a call to action.
  • .btn-info

    A blue button, for quiet actions
  • .btn-success

    A green button, usually used for confirmation actions
  • .btn-warning

    A orange button, can be used for emphazing the consequences of the action.

     
<button class="btn {$modifiers}">Button</button>
